EBV differential expression analysis with DESeq2

Intro

We used DESeq2 to estimate which genes are differentially expressed. For each gene, we used a model of the form:

\[ Y_{ijk} = \mu + \alpha_i + \beta_j + \gamma_{ij} \]

where:

  • \(\alpha_i\) is the effect for the \(i\)th cell, among EBV, EBV- (NOKS) and EBV.dRdZ.

  • \(\beta_j\) is the treatment effect between no_treatment and methyl_cel.

  • \(\gamma_{ij}\) is the interaction term between cell and treatment.

  • \(Y_{ijk}\) are the counts for the \(k\)th replicate of sample with cell \(i\) and treatment \(j\).

Hypothesis tested

Using this framework, for each gene we tested the following hypothesis:

  • Cell effects:

    • EBV effect = EBV- effect (\(\alpha_1 = \alpha_2\))

    • EBV.dRdZ effect = EBV- effect (\(\alpha_3 = \alpha_2\))

  • Cell effects when no_treatment was applied

    • EBV effect = EBV- effect (\(\gamma_{11} = \gamma_{12}\))

    • EBV.dRdZ effect = EBV- effect(\(\gamma_{13} = \gamma_{12}\))

  • Cell effects when methyl_cell was applied

    • EBV effect = EBV- effect (\(\gamma_{21} = \gamma_{22}\))

    • EBV.dRdZ effect = EBV- effect (\(\gamma_{23} = \gamma_{22}\))
