Venn diagram

The Venn diagrams below were constructed by defining a genes to be differentially expressed iff the adjust p.values were below 0.001. First, we compared all genes and noticed that a lot of genes are being differentially expressed regardless of the cell line. However, when the genes are divided by the regulation sign ( i.e. the \(\log_2(\mbox{fold change}))\) sign) we can notice that the same pattern holds, thus an apparent conclusion is that there is not a pattern respect to this sign. I think we should use this to motivate the pathway analysis.

I feel it is a bit odd that there are so many genes being differentially expressed with both cell lines, only NOKs, or only NOKs-AKATA. How are those genes shared?

If we only consider the genes that are differentially expressed (under our definition p.value < 0.001), it seems to be almost the same. If anything the number of meaningful pathways seems to be reduced, producing a more conservative pathway analysis (i.e. probably a bit of signal of non-differentially expressed genes is filtering). However, it is not surprising because the GSEA test works by examining the tails of the signal to noise ratio, which are almost the same between both analysis because the number of differentially expressed genes is very high.

Figure 6A

Scott Fold change: The idea of this figure is two fold. a) Compare the MC over NT fold change between NOKS and EBV cell lines. b) Quantify the ratio between cell fold changes.

For b), we considered the \(y = \beta x + \epsilon\) model, where \(x\) and \(y\) are the \(\log_2 \left(\frac{1 + \mbox{mean MC TPM}}{mean NT TPM}\right)\) for the NOKs, and NOKs-AKATA samples respectively. Below, we calculated the \(H_0: \beta = 1\) test, representing that

Fig 6B (or supplement)

We repeated the previous analysis, but by keeping only the genes that are differentially expressed by cell or treatment (i.e. the ‘Both’ group). In magenta, we observe the slope obtained by considering only the differentially expressed genes, and in orange we observe the linear relationship by restricting the genes by their regulation (i.e. NOKs > 0 and NOKs-AKATA > 0 and NOKs < 0 and NOKs-AKATA < 0 ).
